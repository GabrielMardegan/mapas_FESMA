# mapas_FESMA
Scripts em R dos mapas utilizados na dissertação "Força Estadual de Saúde do Maranhão (FESMA)", com dados sobre o programa Mais IDH, URS atendidas e macrorregiões de saúde.


# Códigos dos mapas usados na dissertação

Este repositório contém os scripts em R utilizados para gerar os mapas apresentados em minha dissertação de mestrado, realizada no Departamento de Ciência Política da Universidade de São Paulo (DCP/USP).

- `map1_maisIDH.R`: Mapa dos municípios atendidos pelo programa Mais IDH (primeiro mandato de Flávio Dino).
- `map2_URS.R`: Mapa das Unidades Regionais de Saúde (URS) do Maranhão, com destaque para as 12 acompanhadas pela FESMA durante segundo mandato de Flávio Dino.
- `map3_macro.R`: Mapa das macrorregionais de saúde do Maranhão, divisão administrativa utilizada pelo programa Planifica Maranhão.

Os scripts utilizam os pacotes `geobr`, `sf`, `ggplot2`, `dplyr` entre outros.
